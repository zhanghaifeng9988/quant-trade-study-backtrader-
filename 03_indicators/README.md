# 模块3：技术指标与信号

脱离玄学，回归量化。本模块不再教你“背诵”图表形态，而是从统计和动量现象的本质出发，探讨经典技术指标的数学底座，并向你展示专业机构的“股票池筛选”与“真伪协整配对”方法。

## 📓 课程目录

| 文件序号 | 内容说明 |
|------|------|
| `01_trend_indicators.ipynb` | 移动平均线（SMA/EMA/MACD）的计算、滞后性与趋势跟踪信号 |
| `02_momentum_indicators.ipynb` | 相对强弱（RSI）、随机指标（KDJ）与价格动量效应 |
| `03_volatility_indicators.ipynb` | 布林带均值回归、ATR 动态止损与 VIX 恐慌指数 |
| `04_universe_selection.ipynb` | 过滤流动性陷阱：构建符合机构交易标准的股票池（Universe） |
| `05_cointegration_correlation.ipynb` | 高相关性 != 可配对。利用 Engle-Granger 检验寻找真实协整关系 |

## 💡 核心认知
- 单一技术指标无法直接构成能长期盈利的系统。
- 流动性过滤是多标的策略的生命线。
