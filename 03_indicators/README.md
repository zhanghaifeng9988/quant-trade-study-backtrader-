# 模块3：技术指标与信号

技术指标是基于历史价格和成交量数据计算的统计量，用于识别趋势、动量和反转信号。

## 内容

| 文件 | 指标 |
|------|------|
| `01_trend_indicators.ipynb` | 简单移动均线(SMA)、指数移动均线(EMA)、MACD |
| `02_momentum_indicators.ipynb` | RSI、随机指标(KDJ/Stochastic)、动量(Momentum) |
| `03_volatility_indicators.ipynb` | 布林带 (Bollinger Bands)、ATR、Keltner Channel |

## 重要提示

> ⚠️ 技术指标本身只是**信号工具**，不是盈利的保证。
> 必须通过严格的**回测**验证信号的有效性！
