# 模块4：策略回测

回测是用历史数据模拟策略执行的过程，是量化交易研究的核心环节。

## 内容

| 文件 | 说明 |
|------|------|
| `01_simple_backtest.ipynb` | 从零手写简单回测引擎（双均线策略） |
| `02_backtrader_intro.ipynb` | 使用 backtrader 框架进行专业回测 |
| `03_performance_metrics.ipynb` | 回测绩效指标全解析 |

## 常见回测陷阱

| 陷阱 | 说明 | 如何避免 |
|------|------|----------|
| **Look-ahead Bias** | 使用了未来数据 | 信号使用 `shift(1)` 入场 |
| **过拟合** | 在样本内表现极好 | 留出样本外测试集 |
| **幸存者偏差** | 只用现存的股票 | 使用历史成分股数据 |
| **忽略成本** | 不计手续费/滑点 | 每笔交易扣除成本 |
| **容量问题** | 忽略流动性约束 | 成交量限制仓位 |
