# 模块7：基本面量化 (Fundamental Quant)

除了从价格的 K 线图里找规律（技术分析），量化投资的另一个巨头派系是基本面量化。我们将公司财报中的会计数据转化为可回测的“因子”，让计算机自动化地进行价值投资。

## 📓 课程目录

| 文件序号 | 内容说明 |
|------|------|
| `01_financial_statements.ipynb` | 财报数据解析：资产负债表、利润表与现金流三大表的量化提权 |
| `02_value_factors.ipynb` | 价值因子（PE, PB, PS 等）的计算、去极值与标准化 |
| `03_quality_factors.ipynb` | 质量因子（ROE, 盈利稳定性，负债率）挖掘优质核心资产 |
| `04_earnings_surprise.ipynb` | 盈余意外（PEAD 效应）与分析师预期数据处理 |

## 💡 基本面量化的力量
基本面因子具有换手率低、容量大、逻辑高度自洽的特点。结合技术面动量因子，是构建稳健多因子 Alpha 策略的核心。
