# 模块9：高频交易与微观结构 (HFT & Microstructure)

为什么你的市价单买入之后立刻就亏损了？冲击成本是如何发生的？高频做市商又如何利用毫秒级的买卖价差持续盈利？本模块揭示市场交易订单流和微观结构的核心机制。

## 📓 课程目录

| 文件序号 | 内容说明 |
|------|------|
| `01_order_book_basics.ipynb` | 限价指令簿（LOB）基础、买卖盘重构与市场微观结构分析 |
| `02_market_making.ipynb` | 做市商存货模型、买卖价差控制策略（Avellaneda-Stoikov 模型基础） |

## 💡 微观逻辑
对于高频交易而言，延迟（Latency）胜过一切天才的模型预判。在这里，你学到的不仅是策略，更有避免“被抢跑（Front-running）”的机构防御艺术。
